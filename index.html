Layout Grids

Layout grids are the foundation of our responsive layout. We are using a 12 column grid. Create a row by using the .grid-row class, and create columns within that row by using the .grid-* classes. The columns can be of any width in number between 1 and 12. As an example, the class .grid-1-2 will create a column of width 50% in a row. We have specific classes for a width of golden ratio as defined in the example below.

You can specify such layout for specific screen sizes as well between small(sm), medium(md), large(lg) and x-large(xl). A class .grid-lg-1-3 will create a container of 33% width on large screens and above. The grids are designed to be small-up which means that by default, a class specified for small screen sizes will apply to larger screen sizes.
